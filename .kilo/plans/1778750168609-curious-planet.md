# Implementation Plan: Enhanced Admission Application for International & Transfer Students

## Overview
Extend the LMS admission system to support:
1. **International students** - Non-Nigerian applicants, alternative qualifications
2. **Transfer students** - Credit transfer from other institutions
3. **Direct entry students** - Year 2+ entry with A-Levels, IB, etc.

## Phase 1: Data Model Extensions

```csharp
private async Task<IEnumerable<DocumentType>> GetRequiredDocumentsAsync(ApplicantType applicantType, Guid programId)
{
    var baseRequired = await dbContext.DocumentTypes
        .Where(t => t.Category == DocumentCategory.Admission && t.IsCompulsory && t.IsActive)
        .ToListAsync();
    
    // Remove Nigeria-specific for international
    if (applicantType == ApplicantType.International)
        baseRequired = baseRequired.Where(t => !t.NigeriaOnly).ToList();
    
    // Add conditional documents
    if (applicantType == ApplicantType.DirectEntry)
        baseRequired.AddRange(await GetDirectEntryDocuments());
    
    // ... more conditions
}
```

## Rollback Plan

- Migration can be reverted using `dotnet ef migrations remove`
- New enum values are additive (non-breaking)
- All new fields are nullable, preserving existing data integrity
